## Проект Компьютерный клуб

### Как собрать проект
    В корневой директории где лежит Makefile, вводим команду make, которая скомпилирует программу с именем program. Можно вручную запустить **./program <file>** или можем ввести **make install** - которая скомпилирует(если требуется) и сразу запустит программу с файлом text.txt.

### как запустить тесты
    **make test** - собирает программу test и запускает тесты.
